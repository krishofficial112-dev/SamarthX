# 📖 Overview

This repository showcases my **UI/UX design work for the SamarthX eGov ecosystem**, a comprehensive education governance platform focused on simplifying digital administration for educational institutions.

The project demonstrates my ability to design complex enterprise applications including dashboards, permission management systems, admission portals, responsive interfaces, authentication systems, landing pages, and communication collateral.

The objective throughout the project was to improve usability, accessibility, information hierarchy, and scalability while maintaining a modern, government-grade user experience.
