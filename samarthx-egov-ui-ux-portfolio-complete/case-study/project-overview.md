# Project Overview

SamarthX eGov is presented here as a portfolio case study covering a broad e-governance product ecosystem. The selected work includes dashboard experiences, role and permission mapping, student admission, authentication interfaces, communication collateral, and long-form website concepts.

## Role

**UI/UX Designer — Krish Garg**

## Responsibilities

- Interpreting product and stakeholder requirements
- Improving information architecture and visual hierarchy
- Designing desktop dashboard experiences
- Creating scalable selection, filtering, and permission workflows
- Designing role-specific login and admission experiences
- Producing responsive visual concepts and communication collateral
- Preparing polished assets for review and developer handoff

## Core Design Direction

The project uses a clean blue-and-white visual language with selective orange, green, and status colors. The system prioritizes readability, structured data presentation, accessible form controls, and a professional government-platform identity.
